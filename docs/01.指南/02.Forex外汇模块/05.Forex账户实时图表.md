## 📈 Forex 账户实时图表功能

**Forex 账户实时图表**功能用于展示交易账户在指定时间范围内的资金变化与风险状态，可视化呈现账户的**余额（Balance）**与**净值（Equity）**走势，帮助交易者直观了解账户盈利能力、回撤情况以及交易风险变化过程。

通过该图表，交易者无需依赖主观感受，而是可以基于真实数据趋势进行复盘、风险评估及策略优化。

---

### 🧩 支持的图表指标

图表支持多种交易相关指标展示，用户可通过顶部选择框自由开启/关闭显示：

| 指标类型 | 描述 |
|----------|------------------------------------------------------------|
| balance | 已结算余额，净利润的累计结果 |
| equity | 动态权益，随市场行情实时波动 |
| volume | 交易持仓量（如果启用显示） |
| profit | 当前浮动盈亏（如果启用显示） |
| freeMargin | 可用保证金（如果启用显示） |
| marginLevel | 保证金水平（如果启用显示） |
| priceAsk / priceBid | 市场卖价与买价（可选） |

> 默认展示核心指标：**Balance & Equity**

---

### 🔍 功能特点

| 功能项 | 说明 |
|--------|-----------------------------------------------------------|
| 时间范围筛选 | 可自定义查询起止日期范围 |
| 动态点位浮窗 | 鼠标悬停可显示精确时间与数值 |
| 多指标选择 | 自由开关显示任意分析指标 |
| 区间缩放 | 可使用底部缩略条快速拖拽放大区域细节 |
| 高频监控 | 图表可实时刷新（交易中更高频更新） |
| 稳定数据呈现 | 支持真实账户及模拟账户无差别显示 |

---

### 📆 查询流程

1. 顶部选择交易账号  
2. 选择交易品种（如 EURUSD、XAUUSD、GBPJPY 等）  
3. 选择起始与结束日期  
4. 点击 **「搜索」**  
5. 图表即时呈现趋势曲线  

---

### 📊 示例说明

图表呈现两条核心曲线：

- **橙色线条（Balance）**：展示从账户起始到当前已结算的稳定资金变化  
- **浅棕色线条（Equity）**：展示随市场实时波动的动态权益变化  

典型可观察到的分析现象包括：

✔ 盈利逐步累积  
✔ 回撤区域及深度  
✔ 策略稳定性及波动率  
✔ 交易执行后的资金节奏  

---

### 🧠 使用建议

- 回测策略或 EA 时优先对比 Balance 与 Equity 的**回撤差值**
- 若长期 **Equity 明显低于 Balance**，说明持仓风险偏高
- 可将图表结果与 **监测规则功能** 或 **交易计划复盘功能** 联动使用
- 如需展示研究报告，可导出图表截图添加至复盘文档

---

### 💡 适用场景

| 场景 | 使用价值 |
|------|--------------------------------------------|
| EA 稳定性检测 | 查看长期曲线是否平滑、无大幅回撤 |
| 手动交易复盘 | 观察决策点前后 Equity 反应 |
| 风险管理 | 判断保证金与权益波动风险 |
| 策略展示 | 用于向合伙人或客户介绍策略表现 |
